# DJ GC77XR — Daily Ritual

A mobile-first daily ritual web experience for DJ GC77XR.

## Features
- SATA to CODA state transition
- Breath timing and ritual prompts
- 118 BPM audio tone option
- Animated particles and geometry
- Daily streak / check-in system using localStorage
- Mobile-first responsive design

## Files
- `index.html`
- `style.css`
- `script.js`

## How to run
1. Upload all files to your GitHub repository.
2. Open `index.html` in a browser or enable GitHub Pages.
3. Tap **Enable Sound** if you want audio.
4. Tap **Start Ritual** to begin.
5. Tap **Check In** after completing the ritual.

## Brand language
- SATA = Sensory Attunement and Tranquility Activation
- CODA = Conscious Observance and Daily Acceptance
- Indigo = quiet state
- Gold fire = activation state
